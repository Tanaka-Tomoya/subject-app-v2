# Subject-app-v2

## Setup

```
$ yarn
```

### client

```
$ cd client
client $ yarn
```

### server

```
$ cd server
server $ yarn
```

## Run

```
$ yarn dev
// This command runs both run commands
```

### client

```
$ cd client
client $ yarn dev
```

### server

```
$ cd server
server $ yarn dev
```
